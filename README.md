# Dotfiles
My dotfiles

WMs: dwm, bspwm

Terminal: st

Shell: zsh

Configs: ~/.config
         
         ~/.repos


# Installation

1. Clone the repository

2. Make the scripts `nopasswd.sh` and `install.sh` executable
  ```
  chmod +x nopasswd.sh
  chmod +x install.sh
  ```

3. Run the scripts 
  ```
  ./nopasswd.sh
  ./install.sh
  ```
